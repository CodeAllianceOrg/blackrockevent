Project Details


Read our full project notes here

Open Referral has developed open protocols for exchanging data about health, human, and social services. A majority of the technology vendors in the field are adopting these protocols — and we now need to develop tools to enable effective federation of data across multiple sources.

This may entail:

  • Transforming sample data sources
  • Deploying the Open Referral API
  • Deploying Dedupe and/or other tools to match records across multiple redundant sources
  • Developing tools to facilitate the resolutions of conflicts and verification of accuracy of data
  • Developing UUID management solution

Objectives:

  • Test multiple Open Referral API deployments
  • Demonstrate that data from multiple sources can be coherently aggregated, visualized, resolved and verified

See this detailed whitepaper about the resource data federation challenge.
